# Web Serial Assistant (Web版纸飞机串口助手) - PRD

## 1. 项目简介
本项目是一个基于 Web Serial API 的网页版串口调试助手，灵感来源于“纸飞机串口助手”。它允许用户直接通过浏览器与硬件设备（如单片机、传感器模块）进行串口通信，实现数据的接收、发送、解析以及实时多通道图表绘制。

## 2. 核心功能需求

### 2.1 串口连接与配置
*   **Web Serial API**: 使用现代浏览器支持的 Web Serial API 进行无插件的串口连接。
*   **波特率选择**: 提供常用波特率下拉列表（如 9600, 115200 等）。
*   **连接/断开控制**: 提供“打开串口”和“关闭串口”的按钮状态切换。

### 2.2 数据解析与处理
*   **协议解析**: 监听接收到的数据流，按行分割，并识别特定格式的数据帧：`{标题}数据1,数据2,数据3...`。
    *   例如：`{IMU}0,-0,0,2,5,0` 或 `{LSM1}-0.469,-0.003,0.963,368.6,385.9,-74.6`。
*   **标题提取**: 动态提取 `{}` 中的名称作为数据通道（Title）。
*   **数据转换**: 将逗号分隔的字符串转换为浮点数数组。

### 2.3 数据可视化 (图表区)
*   **多标签页 (Tabs)**: 当检测到新的 `{标题}` 时，自动在图表区创建对应的标签页（如 `IMU`, `LSM1`, `LSM2`）。
*   **实时折线图**:
    *   X轴：时间（点数或时间戳）。
    *   Y轴：数值。
    *   多线条：同一个标题下的多个数据点分别绘制为不同的线条（Line 1, Line 2...）。
*   **图例与交互**:
    *   提供复选框（Checkbox）用于显示/隐藏特定的数据线。
    *   支持高亮：鼠标悬浮或点击图例/线条时，高亮对应数据线。
    *   联动：点击图表上的数据点，能在下方“原始数据区”定位并高亮对应的原始数据行。

### 2.4 原始数据区 (日志区)
*   **多标签页**: 提供 `main`（显示所有原始日志），以及各个解析出标题的独立标签页。
*   **高亮与联动**: 接收到点击图表事件时，对应行的文本进行高亮显示。
*   **辅助功能**: 
    *   支持文本搜索/过滤（显示TX/显示RX）。
    *   支持一键复制日志数据。

### 2.5 数据发送区
*   **多行输入**: 提供文本域供用户输入需要发送的指令。
*   **发送设置**:
    *   时间戳/超时设置。
    *   循环发送选项。
    *   HEX发送选项。
    *   追加换行选项（回车发送）。
*   **发送与清空按钮**: 触发串口写入操作，以及清空发送区文本。

## 3. 界面布局结构
*   **Top (顶部导航栏)**: Logo, 菜单, 串口号(显示已连接设备), 波特率选择, 打开/关闭串口按钮。
*   **Middle-Top (图表展示区)**: 包含可切换的图表标签页，使用 Recharts 渲染折线图。
*   **Middle-Bottom (数据接收区)**: 包含日志标签页，显示带时间戳和协议的原始字符串，底部带有搜索栏。
*   **Bottom (数据发送区)**: 左侧为多行文本输入框，右侧为发送配置项的复选框及发送/清空按钮。

## 4. 技术栈
*   **框架**: React (Vite, TypeScript)
*   **样式**: Tailwind CSS
*   **图表**: Recharts
*   **图标**: Lucide React
*   **硬件接口**: Web Serial API (`navigator.serial`)
